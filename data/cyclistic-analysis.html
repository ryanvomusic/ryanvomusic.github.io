Cyclistic Bike Share Analysis

Google Data Analytics Capstone Project

Completed: August 2025 SQL • BigQuery • Tableau

Business Task

Analyze usage patterns between annual members and casual riders to identify key behavioral differences that will inform a targeted marketing strategy. The business goal is to convert casual riders into profitable annual members, as finance analysts have determined annual memberships generate significantly higher revenue than single-ride or full-day passes. This analysis will provide data-driven insights to guide marketing campaigns, optimize resource allocation, and maximize membership growth for Cyclistic's bike-share program.

Data Sources

Data Sources

Dataset: Cyclistic historical bike trip data (12 months: August 2024 - July 2024)

Volume: 5,479,059 individual trip records

Source: Public dataset from Motivate International Inc. (real Divvy bike-share data from Chicago)

Key Variables

  • ride_id: Unique identifier for each trip
  • rideable_type: Bike type (classic, electric, docked)
  • started_at / ended_at: Trip start and end timestamps
  • member_casual: User type classification

Calculated Variables

  • ride_length: Trip duration in minutes
  • day_of_week: Extracted from start timestamp

Data Limitations

  • No personally identifiable information (privacy protected)
  • Cannot connect individual trips to specific users
  • Cannot determine if casual riders live in service area or purchased multiple single passes

Methodology

Data Processing and Analysis

Platform: Google BigQuery for data storage and SQL analysis

Data Cleaning: Filtered out negative ride times and data quality issues

Calculated Fields: Created ride_length (minutes) and day_of_week variables

Analysis Approach: Comparative analysis between member_casual groups across multiple dimensions

Key Analyses Performed

  • Usage volume and duration comparisons
  • Daily usage patterns (day of week analysis)
  • Seasonal trends (monthly usage patterns)
  • Bike type preference analysis

Visualization

Tableau for interactive dashboard creation and data storytelling

Quality Controls

  • Removed trips with negative or zero duration
  • Verified data consistency across 12-month period
  • Cross-validated findings across multiple analytical dimensions

Key Findings

Usage Volume vs. Duration Reveals Different Use Cases

  • Members take 76% more rides (3.49M vs 1.99M) but with significantly shorter duration (11.85 min vs 23.60 min average)
  • Short, frequent trips indicate commuting/transportation behavior vs longer recreational rides

Weekly Patterns Confirm Transportation vs. Recreation Usage

  • Members show consistent weekday usage peaking Tuesday-Thursday (commuting behavior)
  • Casual riders peak on weekends with lowest usage midweek (recreational/leisure behavior)

Seasonal Dependency Reveals Different Use Cases

  • Casual riders show dramatic seasonal variation (23K rides in January vs 334K in summer), suggesting recreational/tourism usage patterns
  • Members maintain more consistent year-round usage (112K-465K range), indicating transportation dependency regardless of weather
  • Summer peaks for both groups, but casual riders' 14x variation vs members' 4x variation confirms different motivations

Bike Type Preferences Show Efficiency Focus

  • Both groups prefer electric bikes, but members show slightly stronger preference (64.32% vs 59.20%)
  • Members' higher electric bike usage aligns with efficiency-focused commuting behavior
  • Electric scooter usage minimal but higher among casual riders

Strategic Recommendations

Target Summer & Weekend Marketing Campaigns

  • Focus marketing efforts during June-August when casual riders are most active (334K+ monthly rides)
  • Concentrate on Friday-Sunday when casual usage peaks
  • Leverage tourism seasons and recreational activity periods for maximum reach

Emphasize Convenience & Efficiency Benefits

  • Highlight time-saving advantages of membership for regular users
  • Market electric bike availability and convenience features
  • Position annual membership as the smart choice for consistent users (cost savings over multiple single rides)

Implement Flexible Membership Options

  • Create 3-month seasonal memberships targeting summer tourists and occasional users
  • Offer trial week/month memberships for casual riders showing repeat usage
  • Position seasonal memberships as cost-effective for extended visits or summer recreation
  • Use data to identify casual riders with 2+ months of activity as prime conversion targets

Strategic Recommendations

Business Impact

These recommendations leverage the behavioral data to target casual riders when they're most engaged while addressing their primary usage motivations. Converting even a small percentage of the 1.99M casual riders would significantly impact revenue, given that annual members are substantially more profitable than single-ride users. This analysis provides Cyclistic with a data-driven roadmap for marketing resource allocation and campaign timing based on actual usage patterns rather than assumptions. The distinct behavioral profiles identified enable targeted messaging that speaks to each group's specific needs and usage patterns.
